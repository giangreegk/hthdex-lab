# Hthdex Lab

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于收集、整理和托管多个独立的 HTML 页面。这些页面通常用于实验、演示、工具或简单应用场景，彼此之间没有强关联，各自独立运行。

仓库本身不针对任何特定网站或服务，仅作为页面文件的存储与展示空间。

## 目录说明

```
.
├── pages/          # 存放各独立 HTML 页面及其附属资源（如 CSS、JS、图片等）
│   ├── page-1/     # 单个页面目录示例
│   ├── page-2/
│   └── ...
├── assets/         # 可选：跨页面共享的公共资源（如通用样式、脚本）
└── README.md       # 本文件
```

每个页面放在 `pages/` 下的独立子目录中，方便管理与引用。

## 页面归档说明

- 每个页面均为自包含的 HTML 文件，通常依赖外部资源（如 CDN 库）会注明来源。
- 页面功能可能涉及前端技术实验、小型工具、信息展示等。
- 仓库不保证所有页面长期可用或兼容所有浏览器，部分页面可能仅用于特定环境测试。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修正已有页面。
- 欢迎通过 Issue 提交建议或报告问题，但请注意仓库不面向具体商业或推广目的。
- 如需贡献新页面，请遵循 `pages/` 下的目录结构，并尽量保持文件简洁清晰。
